Malliratkaisu

Tehtävänäsi on tuottaa yllä olevaa malliratkaisua vastaava kuva käyttäen Pixlr X -ohjelmistoa. Tehtäväsi on siis muokata kuvaa sydämen rakenteesta luomalla siihen tekstilaatikkoja ja nuolia, jotka osoittavat sydämen eri osien nimet. Osia tulee olla nimettynä vähintään samat kuin mitä malliratkaisussa näkyy tyhjiä laatikoita. Malliratkaisusta eroten tekstilaatikoissa pitää siis olla tekstilaatikot täydennettynä.

Voit soveltaa ja luoda tekstilaatikot sekä osoittimet haluamallasi tavalla, mutta sinun tulee kuitenkin suorittaa Pixlr:ssa vähintään seuraavia toimintoja:

  • Luoda tekstilaatikko, vaihtaa sen tekstin sekä taustan väriä ja muokata sen sisältämää tekstiä.
  • Luoda Pixlr-elementtejä (esimerkiksi Line), joka toimii osoittimena tietyn sydämen osan kohtaan.
  • Tallentaa kuva png-muodossa 720x1080px kokoisena ja nimellä sydamenosat_EtunimiS (EtunimiS kohtaan oma etunimesi + sukunimen ensimmäinen kirjain).

Lataa ensin alla oleva pohja omalle koneellesi:

Aloita sitten muokkaamaan kuvaa painamalla Pixlr X:n etusivulla Open image-painiketta ja valitsemalla aukeavasta valikosta äsken ladattu kuvapohja. Alta löydät eri muokkaustoimintojen tekemiseen.

Tekstin lisäys kuvaan

Seuraavassa GIF:ssä näkyy vaiheet, kuinka lisätä Pixlr X -ohjelmalla tekstilaatikko sekä vaihtaa laatikon värejä ja tekstiä.

Vaiheet tekstilaatikon lisäämiseen ovat seuraavat:

  1. Valitse sivuvalikosta ADD TEXT.
  2. Valitse aukeavasta valikosta Add new text - New default text.
  3. Vaihda tekstin väri muokkaamalla Fill-ominaisuutta. Voit valita eri värin kuin mallissa
  4. Vaihda taustan väri aukaisemalla Background-valikko ja valitsemalla siihen uusi väri. Voit valita eri värin kuin mallissa.
  5. Raahaa laatikko oikealle paikalle hiiren vasenta näppäintä pohjassa pitäen.
  6. Muokkaa tarvittaessa laatikon kokoa sen reunoista hiirellä vetämällä.
  7. Muokkaa tekstiä tuplaklikkaamalla laatikkoa tai muokkaamalla vasemmasta valikosta Text-ominaisuutta.

Elementin lisäys kuvaan

Seuraavassa GIF:ssä näkyy vaiheet, kuinka lisätä kuvaan Pixlr-elementti ja kuinka vaihtaa sen kokoa sekä paikkaa.

Vaiheet Pixlr-elemntin lisäämiseksi ovat seuraavat:

  1. Valitse sivuvalikosta ADD ELEMENT.
  2. Valitse Shape.
  3. Valitse LINE.
  4. Valitse valikosta haluamasi viiva. Mallissa tämä on valkoinen viiva.
  5. Muokkaa elementtiä sopivan kokoiseksi laatikon kulmasta vetäen.
  6. Siirrä elementtiä, kuten tekstiälaatikkoa eli vasen näppäin pohjassa raahaten.
  7. Käännä tarvittaessa viivaa laatikon päällä olevasta pallosta.

Tallentaminen

Seuraavassa GIF:ssä näkyy vaiheet, kuinka tallentaa kuva tietyllä tiedostotyypillä, koolla ja nimellä.

Vaiheet tallentamiseen ovat seuraavat:

  1. Paina SAVE-näppäintä.
  2. Vaihda nimi kohdasta File name.
  3. Vaihda File type PNG-tyyppiseksi.
  4. Vaihda leveys (Image width) 720px kokoiseksi ja korkeus (Image height) 1080px kokoiseksi. Näistä toista muokkaamalla toisen pitäisi skaalautua automaattisesti.
  5. Paina Download ja muokatun kuvan pitäisi latautua tietokoneellesi.
